Bahasa Pemrograman JavaScript

Javascript adalah bahasa yang berfungsi untuk membuat script â€“ script program yang dapat dikenal dan dieksekusi oleh web browser dengan tujuan untuk menjadikan halaman web lebih bersifat interaktif. Javascript bersifat client side programming language. Client side programming language adalah tipe bahasa pemrograman yang pemrosesannya dilakukan oleh client. Aplikasi client yang dimaksud merujuk kepada web browser seperti google chrome dan mozilla firefox.

Fungsi javascript untuk membuat interaksi antara user dengan website menjadi lebih cepat tanpa harus menunggu pemrosesan di web server. Sebelum javascript, setiap interaksi dari user harus diproses terlebih dahulu oleh web server.
